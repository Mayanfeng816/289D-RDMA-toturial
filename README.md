# 289D-RDMA-toturial
This project is about RDMA tutorial in the GPU era.
We have used 
[auto-deploy github pages ](https://kyli-leo.github.io/289D-RDMA-toturial/)
to have a better visualization.

It includes following parts:
- RDMA Basics
    - Introduction to RDMA
    - RDMA Verbs API
    - Memory Registration
    - Queue Pairs
- GPU direct RDMA
- Code Examples
    - RDMA Read/Write Example
    - One-sided vs Two-sided Test
    - RC vs UD Test
    - RW GPU Steps
    - UCCL optimizations
    - AWS EFA
